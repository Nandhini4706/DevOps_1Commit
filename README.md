# DevOps First Commit Project

## Project Description
This project contains basic Java files created for practicing Git and GitHub.

## GitHub Connection
This repository is connected using GitHub Personal Access Token (PAT).

Connection Type: HTTPS  
Authentication: Token-based

## Commands Used in Aws
sudo su
apt install git
git init

touch Dev1new{1..10}.java

## Commands Used for git connection
git add .  
git commit -m "My First DevOps Commit"  
git remote add origin <repository-url>  
git push -u origin main
